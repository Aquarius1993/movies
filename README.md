	1. npm

	3. grunt

### 安装包
	npm instal
	
### 安装mongodb
	Mac上安装mongodb:
		1.更新brew： brew update
       	2.安装mongodb:brew install mongodb
        3.启动： mongod --config /usr/local/etc/mongod.conf
        4.再打开：mongo
